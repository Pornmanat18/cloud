• การประมวลผลแบบคลาวค์
  • รูปแบบบริการคลาวด์
  • รูปแบบการติดตั้งใช้งาน
  • ข้อดีและข้อจำกัดของการประมวลผลแบบคลาวด์
  • คลาวด์ในงานเขียนโปรแกรม

รายวิชานี้ศึกษาหลักการ แนวคิด และโครงสร้างพื้นฐานของการประมวลผลแบบคลาวด์ รวมถึงรูปแบบการให้บริการ ได้แก่ โครงสร้างพื้นฐานในรูปแบบบริการ (Infrastructure as a Service: IaaS), แพลตฟอร์มในรูปแบบบริการ (Platform as a Service: PaaS), และซอฟต์แวร์ในรูปแบบบริการ (Software as a Service: SaaS) ตลอดจนรูปแบบการติดตั้งใช้งานคลาวด์ ได้แก่ Public Cloud, Private Cloud และ Hybrid Cloud

เนื้อหายังครอบคลุมถึงการพัฒนาแอปพลิเคชันบนคลาวด์ การใช้งาน Cloud API, การใช้บริการแบบไร้เซิร์ฟเวอร์ (Serverless Computing), การทำงานร่วมกับระบบ DevOps และกระบวนการพัฒนาและปรับใช้ซอฟต์แวร์แบบต่อเนื่อง (CI/CD) พร้อมทั้งวิเคราะห์ข้อดี ข้อจำกัด ความปลอดภัย และแนวโน้มของเทคโนโลยีคลาวด์ในปัจจุบัน โดยมุ่งเน้นการประยุกต์ใช้งานจริงในการเขียนโปรแกรมและพัฒนาระบบ
